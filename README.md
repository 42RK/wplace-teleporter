# wplace-teleporter
A wplace navigation tool letting users teleport to any pixel of any chunk at a desired zoom level.
</br>
The webtool supplements the lacking built in favorites fuction with a teleport log and favorites stored locally.
</br>
It also supports exporting and importing favorites from .json files for sharing with friends or switching between browsers.
